# AgriTalk Backend (Spring Boot API)

The AgriTalk Backend is a **RESTful API** built using **Spring Boot + Hibernate + MySQL**.  
It handles farmer registration, farm management, crop details, tasks, notifications, and integrates with the **Rasa Agent** to support AI-driven agricultural assistance.

---

## ⭐ Features

- 👨‍🌾 Farmer registration & login
- 🧑‍🌾 Multiple farms per farmer (Multitenant-style structure)
- 🌱 Crop management & seasonal recommendations
- 📅 Farm tasks & reminders
- 💰 Expense & revenue tracking
- 🔄 API integration with Rasa for chatbot output
- 🕌 Clean 3-layer architecture (Controller → Service → Repository)

---

## 🛠️ Technology Stack

- **Java 17+**
- **Spring Boot**
- **Spring Web**
- **Spring Security / JWT**
- **Hibernate / JPA**
- **MySQL**
- **Lombok**
- **Maven or Gradle**
- **Spring Validation**
